# AiHubMix - 企业级 AI 模型 API 中转服务

欢迎使用 **AiHubMix** —— 专业的 **AI 模型 API 中转服务**！

我们基于标准的 OpenAI API 格式，支持 OpenAI 官方所有模型，以及 Claude、Gemini、DeepSeek、阿里 Qwen 等主流大模型。只需修改模型名称，就能轻松切换，无需改代码。

## 公司背景与保障

- **主体公司**：AiHubMix, LLC（美国）
- **官方授权**：微软 Azure、AWS、GCP 官方合作伙伴
- **服务保障**：已为多家知名应用提供稳定生产环境服务

## 独家优势：OpenAI + Azure 双端点

AiHubMix 会同时连接 OpenAI 官方和微软 Azure OpenAI，并智能把部分请求路由到 Azure 进行自动内容审核。

**好处**：
- 安全性更高（Azure 有自动过滤）
- 更合规：请不要请求暴力、政治、色情等敏感内容
- 稳定性更好

## 快速上手（3 分钟完成）

1. **充值**：进入充值页面，充一点余额（用多少充多少）
2. **创建 API Key**：去 API Keys 页面，一键创建并复制你的 Key（以 `sk-` 开头）
3. **替换地址**：
   - 稳定地址：`https://aihubmix.com`
   - 备用地址：`https://api.aihubmix.com`

**小提示**：开发者把 `https://api.openai.com` 换成上面地址即可。支持 Lobe-chat、Cursor、LangChain 等工具。
